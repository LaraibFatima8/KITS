Gesture Music Controller

Intermediate

Transform your PLanzo into a wireless music controller! Control volume, skip tracks, and play/pause music using intuitive hand gestures.

👆
👇
👈
👉

Gestures: Up/Down = Volume, Left/Right = Track Skip

// Music Controller Code
void handleMusicGestures() {
  uint8_t gesture = apds.readGesture();
  
  switch(gesture) {
    case APDS9960_UP:
      volumeUp();
      showOLED("♪ Volume UP");
      break;
    case APDS9960_DOWN:
      volumeDown();
      showOLED("♪ Volume DOWN");
      break;
    case APDS9960_LEFT:
      previousTrack();
      showOLED("⏮ Previous");
      break;
    case APDS9960_RIGHT:
      nextTrack();
      showOLED("⏭ Next");
      break;
  }
}

Pro Tip:

Connect your PLanzo to Spotify using their Web API for real-time music control. You can even display the current song name on the OLED!

Advanced Productivity Tracker

Advanced

Take your PLanzo to the next level with advanced productivity tracking. Monitor work sessions, break times, and create detailed analytics of your day.

Enhanced Features:

📊 Real-time Productivity Graphs

⏰ Smart Break Reminders

📱 Mobile App Integration

☁️ Cloud Data Sync

// Productivity Analytics
void trackProductivity() {
  static unsigned long sessionStart = 0;
  static int pomodoroCount = 0;
  
  uint8_t gesture = apds.readGesture();
  
  if(gesture == APDS9960_UP && sessionStart == 0) {
    sessionStart = millis();
    showOLED("🎯 Focus Started");
    logActivity("FOCUS_START");
  }
  
  if(gesture == APDS9960_DOWN && sessionStart > 0) {
    unsigned long sessionTime = millis() - sessionStart;
    pomodoroCount++;
    showOLED("✅ Session " + String(pomodoroCount));
    saveToCloud(sessionTime, pomodoroCount);
    sessionStart = 0;
  }
}

Analytics You Can Track:

  • Daily focus hours and patterns
  • Most productive times of day
  • Break frequency optimization
  • Weekly/monthly productivity trends

Digital Art Canvas

Intermediate

Turn your PLanzo into a digital art tool! Draw patterns, create animations, and generate art based on your hand movements.

🎨

Gesture → Art!

👆 = Line Up
👇 = Line Down
👈 = Line Left
👉 = Line Right
// Gesture Art Generator
void createArt() {
  uint8_t gesture = apds.readGesture();
  static int x = 64, y = 32; // Center of OLED
  
  display.clearDisplay();
  
  switch(gesture) {
    case APDS9960_UP:
      drawLine(x, y, x, y-10);
      y -= 10;
      break;
    case APDS9960_DOWN:
      drawLine(x, y, x, y+10);
      y += 10;
      break;
    case APDS9960_LEFT:
      drawLine(x, y, x-10, y);
      x -= 10;
      break;
    case APDS9960_RIGHT:
      drawLine(x, y, x+10, y);
      x += 10;
      break;
  }
  
  display.display();
}
